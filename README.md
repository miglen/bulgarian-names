# 🇧🇬 Bulgarian Names Generator

A comprehensive collection of Bulgarian first names (male & female) and family names, with a simple web-based random name generator.

🌐 **Live Demo:** [names.miglen.com](https://names.miglen.com)

## Features

- **27 000+ Bulgarian names** - extensive collection of authentic Bulgarian names
- **Random name generation** - generate random Bulgarian names with one click
- **Gender selection** - choose male, female, or random
- **Copy to clipboard** - easily copy generated names
- **Mobile-friendly** - responsive design works on all devices
- **No dependencies** - pure HTML, CSS, and JavaScript

## Dataset

The name database includes:

| Category | Count |
|----------|-------|
| Male first names | 8000+ |
| Female first names | 7000+ |
| Male family names | 6000+ |
| Female family names | 4000+ |

## Usage

### Web Interface

Simply visit [names.miglen.com](https://names.miglen.com) and click "Generate" to get a random Bulgarian name.

### Local Development

```bash
# Clone the repository
git clone https://github.com/miglen/bulgarian-names.git

# Open in browser
open index.html
```

### Using the Data

The names are stored in JavaScript arrays in `index.html`:

```javascript
const firstNames = {
  "male": ["Александър", "Борис", "Георги", ...],
  "female": ["Анна", "Мария", "Елена", ...]
};

const familyNames = {
  "male": ["Иванов", "Петров", "Георгиев", ...],
  "female": ["Иванова", "Петрова", "Георгиева", ...]
};
```

# 🇧🇬 Генератор на български имена

Обширна колекция от български имена (мъжки и женски) и фамилни имена, с уеб-базиран генератор на случайни имена.

🌐 **Демо:** [names.miglen.com](https://names.miglen.com)

## Функционалности

- **27 000+ български имена** - обширна колекция от автентични български имена
- **Генериране на случайни имена** - генерирайте случайно българско име с един клик
- **Избор на пол** - изберете мъжко, женско или случайно
- **Копиране** - лесно копиране на генерираните имена
- **Мобилна версия** - адаптивен дизайн за всички устройства
- **Без зависимости** - чист HTML, CSS и JavaScript

## База данни

Базата данни с имена включва:

| Категория | Брой |
|-----------|------|
| Мъжки имена | 8000+ |
| Женски имена | 7000+ |
| Мъжки фамилии | 6000+ |
| Женски фамилии | 4000+ |
